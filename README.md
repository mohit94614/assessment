# Gentelella-laravel
AdminBSB Material Design Template in Laravel 5.6


###Installation

#clone 
     ```sh
     git clone
     ```
    
#Installation
    ```sh
    npm install
    ```
     ```sh
    composer update --prefer-source
     ```
     
#Setup
    ```sh
    cp .env.example .env
    php artisan key:generate
     ```
     
#Running
    ```sh
    php artisan serve
    ```
